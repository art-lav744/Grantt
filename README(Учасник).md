# 🏆 Grantt

## 📌 Опис
Grantt — це платформа для проведення програмного турніру, де учасники виконують завдання (подібно до e-olymp) та надсилають свої рішення у вигляді коду.

Система дозволяє:
- виконувати завдання
- подавати рішення через GitHub
- отримувати оцінку від журі
- переглядати результати

---

## 🌐 Деплой

Система може бути розгорнута:

- Gunicorn + Nginx
- Render / Railway
- VPS сервер

Основні кроки:
- Налаштування бази даних (PostgreSQL)
- Запуск через Gunicorn
- Налаштування Nginx

---

## ⚠️ Технічні обмеження

- Використання Python та Django
- Використання ООП
- REST API архітектура
- Унікальність email користувача
- Обов’язкова валідація даних
- Робота через GitHub

---

## 📤 Як подати рішення

Учасник повинен:

1. Виконати завдання
2. Завантажити код у GitHub
3. Надати:
   - посилання на репозиторій
   - (за потреби) demo відео
4. Переконатися, що код запускається

---

## 📊 Як оцінюється рішення

Журі оцінює роботи за такими критеріями:

- ✅ Коректність (код працює)
- ✅ Відповідність умовам задачі
- ✅ Якість коду (читабельність, структура)
- ✅ Оптимальність рішення
- ✅ Наявність README

### Додатково:
- ⭐ Оригінальність
- ⭐ Якість demo

---

## 🧪 Must Have (обов’язкові умови)

Робота НЕ оцінюється, якщо:

- [ ] Відсутнє GitHub посилання
- [ ] Код не запускається
- [ ] Порушено умову задачі
- [ ] Пропущено дедлайн
