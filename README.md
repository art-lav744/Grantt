🎯 GRANTT — СИСТЕМА УПРАВЛІННЯ ТУРНІРАМИ ТА ГРАНТАМИ

Grantt — це сучасна Full-stack платформа для проведення ІТ-турнірів, хакатонів та грантових програм. Система дозволяє учасникам створювати команди та подавати роботи, журі — оцінювати проєкти за критеріями, а організаторам — повністю контролювати етапи проведення змагань.

Статус проєкту: Всі основні модулі (Backend + Frontend) інтегровані та функціонують.

ТЕХНОЛОГІЧНИЙ СТЕК
Backend: Python 3.10+ / Django / Django REST Framework (DRF)
Frontend: Angular 17+ (Standalone Components, RxJS)
База даних: SQLite (для розробки) / PostgreSQL (для продакшну)
Автентифікація: JWT (JSON Web Tokens)

ШВИДКИЙ ЗАПУСК

1. Підготовка репозиторію
Команда: git clone https://github.com/art-lav744/Tournament_2026.git
Команда: cd Tournament_2026

2. Запуск Backend (API)
Бекенд є обов'язковим для роботи системи, оскільки він обслуговує всі запити фронтенду.
Команди для налаштування:
python -m venv venv
venv\Scripts\activate (для Windows)
pip install -r requirements.txt
py manage.py migrate
py manage.py create_admin
py manage.py setup_admin
py manage.py seed_data
py manage.py runserver
API буде доступне за адресою: http://127.0.0.1:8000

3. Запуск Frontend (Angular)
Фронтенд є основною частиною взаємодії з користувачем.
Команди для налаштування:
cd frontend
npm install
npm start
Інтерфейс буде доступний за адресою: http://localhost:4200

ОСНОВНИЙ ФУНКЦІОНАЛ

Користувачі та ролі:
Учасник: Реєстрація, створення або вступ у команду, завантаження робіт.
Журі: Перегляд поданих робіт, виставлення оцінок за бальною системою.
Організатор або Адмін: Створення турнірів, керування раундами, розподіл робіт між журі.

Турніри:
Динамічні статуси (Реєстрація, В процесі, Завершено).
Система команд: капітани можуть додавати учасників за Email.
Автоматичний підрахунок середнього балу на основі оцінок журі.
